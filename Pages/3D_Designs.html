Background Story

Building an enclosure for a product or circuit board was one of the challenges that I always had in the past. During my master's degree, there was an extra activity class for designing a glider using Fusion360. I found out about this class a bit late and joined the class in the third week. When I joined the class, I was feeling so confused because of missing the first sessions. It was quite disappointing for me to see other students in class quickly create 3D models while I was struggling to understand the basic 3D modelling terminology.
The only thing I learned from the class was the name of the software. I decided to learn it at home without attending classes.

Summary of My 3D Designs

I have done a large number of projects to improve my 3D designing and modelling skills. Some of my 3D designs can be 3D printed and used as a functional product. One of the interesting parts of the designs is creating a 3D animation of each model to present how all the pieces are attached to create the final model. I usually make the animation for the major projects that are described below.

Robotic Arm

This is a robotic arm with 4 axes of movement (4 motors). This design is created by simple pieces however throughout this project I learned how to use joint features in Fusion360. Below is a short video of this design.


Humane Animal Trap

This is an animal trap that I designed to catch and release undesired animals without any harm. For instance, the trap can be used for small mice and lizards. I have implemented a one-way gate inside the trap that opens up when the animal falls into it and stays closed while the animal is inside the box. The animal can be released safely and unharmed using the sliding door at the bottom.


Love Sphere

This model was quite interesting with a fair bit of complexities. Initially, I did not have a plan to print this model so ignored all the 3D printing details. To be able to print the model I had to make some changes to the design such as creating, splitting bodies, and adjusting overhangs.
Below is a short video of this design.

Final Words

Learning 3D designing and printing opened up a big door for me. Fusion360 has significantly helped me during my Master's degree and work to export my project ideas into the 3D world.
In my opinion, 3D printing is far more challenging than 3D designing and knowing the capabilities and specifications of the printer plays a key role in the print quality.
All in all, I love to brainstorm about an enclosure or housing for my circuit and be able to design and print it out.
Happy 3Ding :)

Sudoku Solver
Using LabVIEW

I love solving sudoku and finding new strategies to do it faster.
I started this project as my latest challenge, which is still in progress


Background Story

Solving sudoku is one of my hobbies, and I love challenging myself with super difficult sudoku. When I started solving sudokus, probably 15 years ago, I was so excited to explain it to my siblings and parents. My siblings loved the challenge, and we started solving sudokus every day from magazines and sudoku booklets. I got better and better at solving sudokus every day.

I learned about LabVIEW a couple of years ago and since then my mind has been thinking about doing more projects with this powerful software. I decided to make a LabVIEW project that can get sudoku from a user and solve it quickly.

Key Features

This LabVIEW application should run and allow the user to enter all the available values for each cell then perform a sanity check to ensure the entered values are correct and there are no duplicate numbers in rows, columns, and small squares. In the context of the user interface, start and stop buttons will be added as well as a 9x9 sudoku.

Challenges and Solutions

It seems easy to do this project however it gets more difficult when more solving strategies are added to the code. Working with arrays multi-dimension arrays could be challenging as well because of the complexity of reading and writing into each element of the array.
All the solving strategies that I have used are listed below
- Striping sudoku into rows, columns, and 3x3 squares (boxes)
- Find single possible numbers in each row, column, and box.
- Find two cells with the same possible two-value numbers in row, column and box
- Find three cells with the same possible three-value numbers in row, column and box
- Evaluate all the possibilities and confirm single-value cells

Final Words

This is an ongoing project and I am still working on some of the solving strategies. The main complexity of this project is around implementing the new strategies without affecting the previous strategy. Other than that, this project helped me a lot with working with arrays and clusters in LabVIEW.
There are a lot of interesting development opportunities for this project such as implementing image recognition to capture an image of a sudoku and provide the solution or adding buttons and more controls to allow the user to solve the sudoku.
